# qa-test-makers
cypress + api + funcional test

## Cómo ejecutar

1. Clonar el repositorio
git clone https://github.com/GabrielaMachado/qa-test-makers

2. Instalar dependencias
npm install

3. Ejecutar tests en modo headless
npx cypress run

4. Abrir Cypress UI (opcional)
npx cypress open
